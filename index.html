GITHUB

é uma app que te deixa contruir e destribuir software

em uma só platafroma, colaborativa

-QUE PODE FAZER O GITHUB?

Github pode te ajudar a dar "host"(ter um dominio)

teres um site a funcionar, mas a que custos?

Github como pode dar host a pagar e sem pagar.

-QUAL É A DIFERENÇA DE DAR HOST NO TEU SITE NO GITHUB A PAGAR E SEM PAGAR?

Github como já foi explicado pode dar host para os teus sites

o principal objetivo deste site ou aplicação dependendo da situação

para guarda ficheiros teus e informações tuas como projetos html,css e c++ ente outros.

esta explicação toda é para dizer que quando estás a pagar ao Github para dar host no teu site

teus ficheiros não estarão publicos mas se fores sem pagar o teu projeto vai estar publico e todos podem no usar e alterar.

-COMO É QUE O GITHUB GUARDA E ORGANIZA OS TEUS FICHEIROS E DECUMENTOS?

github usa o sistema git criado por Linus Torvald o mesmo criador do kernel do linux

sistema git foi criado para o objetivo de organizar e monitorar o seu projeto que estava em desenvolvimento neste caso o kernel do linux.

-PARA QUE SERVE O GITHUB EM GERAL?

O Github serve para ajudares pessoas com os seus problemas, ajudar pessoas a conseguirem coisas que querem como um jogo

ou programas que estão a precisar e alguem criou.

Em geral o Github é uma comunidade de ajudas e pessoas que querem concretizar sonhos.
